
#Sistema Adição ADM
Adicionar Jogo: 
[Nome do jogo] 
[Imagem de capa] 
[Desenvolvedora] 
[Publicadora] 
[Plataformas] 
[Tempo médio de gameplay] 
[Gêneros]

Modal de cadastro:

textinput nome do jogo
file input arquivo de imagem de capa
textinput nome Desenvolvedora
textinput nome Publicadora
dropdown com plataformas (multiseleção)
dropdown Gêneros (multiseleção)
    
Modal Adição a lista

textinput (auto complete com os jogos cadastrados)
textinput[formatação de tempo (hora/minuto/?segundo?)] (opcional)
dropdown [seleção unica de nota 1-10]
dropdown plataforma jogada [baseada pelo cadastro]
dropdown seleção lista [Falta jogar/zerado/platinado/platinar/dropado/?lista de desejos?]

#Sistema Adição Usuário 
[Nome do jogo] (Box de seleção com a pesquisa pelo nome exibindo nome imagem de capa)
[Listar dados do jogo] (Confirmação de dados pelo usuário)
[Adicionar tempo de jogo] (Tempo que levou para o usuário completar o jogo)
[Adicionar Nota de jogo] (Seleção de nota com slide de seleção entre 1 e 10)
[Adicionar Plataforma jogada] (Caixa de seleção entre as plataformas disponíveis do jogo, em caso de apenas uma ser cadastrada jogo será automaticamente adicionado aquela plataforma)
[Selecionar tipo da lista] (Check box com tipo de lista que vai ser adicionada[Para Jogar|Jogando|Zerei|Platinei|Para Platinar|Dropei|Lista de desejos???])
